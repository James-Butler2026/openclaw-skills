# meme-architect

Context-Aware Meme Generator für OpenClaw.

## Installation

```bash
git clone https://github.com/James-Butler2026/openclaw-skills.git
cd openclaw-skills/meme-architect
pip install Pillow requests
```

## Nutzung

```bash
# Automatische Template-Auswahl
python3 scripts/meme_architect.py "Mein Code läuft endlich!"

# Spezifisches Template
python3 scripts/meme_architect.py "Feature Request um 17:55" --template two-buttons
```
