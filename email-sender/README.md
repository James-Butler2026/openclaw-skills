# email-sender

name: email-sender description: Send emails via SMTP using configured accounts (Web.de, Gmail, etc.). Supports plain text emails with custom subject and body. Uses credentials from .env file. Perfect for quick notifications, alerts, and automated messages. Einfaches Versenden von E-Mails via SMTP. Nutzt konfigurierte Accounts aus der .env Datei.
